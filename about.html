О проекте

Гид по японской мифологии

Добро пожаловать в мир, где мифы обретают форму. Сочетание минималистичного дизайна и дополненной реальности позволяет не просто узнать о существах, но и увидеть их в своём пространстве.


Технологии

1. HTML5, CSS3 (Flexbox/Grid)
2. Vanilla JavaScript
3. Адаптивный дизайн
4. JSON для хранения данных о существах
5. Git / GitHub (управление версиями и хостинг)

Дизайн-концепция

Концепция сайта построена на контрастах: глубокий тёмный фон символизирует таинственность японских легенд, а красные акценты отсылают к традиционным тории, печатям и божественным атрибутам.
