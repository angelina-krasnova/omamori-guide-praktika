# Omamori Guide

Гид по японской мифологии

## О проекте
Сайт-путеводитель по миру ёкаев, ками и древних японских легенд. 
Проект выполнен в рамках производственной практики

## Технологии
- HTML5, CSS3 (Flexbox/Grid, анимации)
- Vanilla JavaScript (динамическая генерация контента)
- JSON для хранения данных о существах
- Git / GitHub Pages (хостинг)
- Адаптивный дизайн (mobile-first подход)
